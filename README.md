### How It Works

1. **Citizens** submit grievances via a simple web interface.
2. **Agencies** receive assigned issues with clear SLA timelines.
3. Each state change (created, in-progress, resolved, overdue) **updates the VSI**.
4. **All records** are:
   - Traceable  
   - Auditable  
   - Publicly visible in a dashboard
